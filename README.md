How to Run Locally
node version: v22.14.0

Frontend
check .env.example file
Add and edit your .env file with your backend API URL.
git clone https://github.com/tezcanTunahan/playable_frontend.git
cd playable_frontend/
npm install
npm run dev
It will run on http://localhost:3000.

Backend
You need a MongoDB database to access. You can create one on your VPS or via cloud.mongodb.com and set up your .env file as per .env.example.
git clone https://github.com/tezcanTunahan/playable_backend.git
cd playable_backend
npm install
npm run dev
